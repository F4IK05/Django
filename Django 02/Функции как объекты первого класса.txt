Functions as first-class objects - свойство языка, при котором функции можно использовать так же, как числа, строки или любые другие переменные.


Можно:

 - Присвоить функции переменным.
 - Передавать их в качестве аргументов другим функциям.
 - Возвращать их из функций.
 - Храните их в структурах данных, таких как списки или словари.



1. Присваивание функций переменным

def msg(name):
    return f"Hello, {name}!"

# Присваивание функции переменной
f = msg

# Вызов функции с использованием переменной
print(f("Emma")) # Hello, Emma!


- Функция msg присваивается переменной f.
- Теперь функцию f можно использовать для вызова сообщения (msg), демонстрируя, что функции ведут себя как переменные.



2. Передача функций в качестве аргументов

def msg(name):
    return f"Hello, {name}!"

def fun1(fun2, name):
    return fun2(name)

# Передача функции msg в качестве аргумента
print(fun1(msg, "Alex")) # Hello, Alex!


- Функция fun1 принимает на вход другую функцию (fun2).
- Сообщение передается в fun1, который затем вызывает его с именем "Alex".


3. Возврат функций из других функций

def fun1(msg):
    def fun2():
        return f"Message: {msg}"
    return fun2

# Получение функции fun2 из fun1
func = fun1("Hello, World!")
print(func()) # Message: Hello, World!


- Функция fun1 определяет другую функцию fun2 и возвращает её.
- Функция func сохраняет возвращаемую функцию fun2, которую можно выполнить позже.


4. Хранение функций в структурах данных
def add(x, y):
    return x + y

def subtract(x, y):
    return x - y

# Хранение функций в словаре
d = {
    "add": add,
    "subtract": subtract
}

# Вызов функций из словаря
print(d["add"](5, 3)) # 8
print(d["subtract"](5, 3)) # 2

- Функции сложения и вычитания хранятся в словаре.
- Доступ к ним осуществляется с помощью соответствующих ключей, и они выполняются напрямую.

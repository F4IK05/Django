2. Генераторы

Generators - это специальные функции, которые возвращают значения по одному, а не сразу все.
Любая функция содержащая yield является генераторной функцией.

При вызове генераторная функция возвращает генератор-итератор или просто генератор


def count_up_to(n):
    i = 1
    while i <= n:
        yield i
        i += 1

num = count_up_to(5)
print(next(num))  # 1
print(next(num))  # 2
print(next(num))  # 3
print(next(num))  # 4
print(next(num))  # 5
print(next(num))  # StopIteration


Они позволяют экономить память, потому что значения создаются только тогда, когда они нужны.


Как работает генератор:
 - При создании генератора стартовая позиция находится в самом начале генератора
 - При вызове метода next генератор выполняется до тех пор, пока не встретит yield
 - Встретив yield генератор возвращает значение записанное в переменную i и запоминает позицию
 - Вызываем еще раз метод next. Генератор снова выполняется до следующего yield, возвращает следующее значение и снова запоминает позицию.
 - Так продолжается до тех пор, пока генератор не опустеет или не встретит ключевое слово return


Разница между return и yield

return:
 - Завершает функцию
 - Возвращает одно значение
 
yield:
 - приостанавливает функцию 
 - запоминает ее состояние
 - при следующем вызове продолжает работу
